# Texas Child Mental Health Care Consortium (TCMHCC) Website

The public website for the **Texas Child Mental Health Care Consortium (TCMHCC)**, a statewide initiative connecting child mental health specialists with primary care providers and communities across Texas.

## Overview

TCMHCC coordinates child and adolescent mental health care across Texas through a network of academic health centers. This site serves as the public-facing hub for the consortium, providing information about:

- Program overview and organizational structure
- Active initiatives and collaborative programs
- Executive committee member directory with headshots and bios
- Meeting archive — agendas and minutes from 2019 through mid-2020
- COVID-19 mental health resources (English and Spanish)

## Site Sections

| Section | Content |
|---------|---------|
| Home | Program overview and leadership |
| Overview | TCMHCC mission, structure, and diagram |
| Initiatives | Active programs and strategic priorities |
| Meetings | Agendas, minutes, and handouts (Sept 2019 – July 2020) |
| Executive Committee | Member roster with headshots |
| Resources | COVID-19 coping guides, telehealth consent forms, CPAN participation agreements |

## COVID-19 Resources

The resources section includes materials produced in response to the 2020 pandemic: patient consent forms for telemedicine visits (English/Spanish), coping guides, and HHSC mental health support line information.

## Tech Stack

- **Vanilla HTML / CSS / JavaScript** — no framework
- **Responsive CSS** — per-section stylesheets with mobile breakpoints
- **jQuery** — navigation and interactive behaviors (`assets/js/init.js`)
